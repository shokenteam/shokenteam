# SHOKEN

**SHOKEN** is a professional non-custodial trading platform focused on real-time markets, transparent execution, and clean market structure.

We are building a unified trading terminal that brings together:
- Perpetual Futures
- Prediction Markets
- CLOB-based Spot Trading (coming soon)

All designed with a strong emphasis on **market mechanics, orderbooks, and execution clarity**.
